# TicketVault API

A simple ticket tracking REST API built with FastAPI.

## Features
- Create and list support tickets
- RESTful API design
- Pydantic data validation
- In-memory storage (for demo purposes)

## Tech Stack
- Python 3
- FastAPI
- Pydantic

## How to Run
```bash
pip install fastapi uvicorn
uvicorn app.main:app --reload
